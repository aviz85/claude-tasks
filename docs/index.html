Claude Tasks

A GTD-based personal task management system powered by Claude Code. Capture, clarify, organize, and execute with AI assistance.

Built with Claude Code Skills, Commands & Agents

/add - Instant Capture

Brain dump anything. Natural language parsing extracts priority, time estimates, projects, and tags automatically.

/standup - Daily Planning

Start each day with clarity. Review yesterday, process inbox, select your top 3 focus tasks.

/focus - Deep Work Mode

Single-task focus sessions. New ideas go to inbox. Stay in the zone.

/review - Weekly Retrospective

Friday ritual. Analyze patterns, clean up stale tasks, plan next week's priorities.

Eisenhower Matrix

Automatic prioritization using urgent/important quadrants. Never wonder what to work on next.

Smart Estimation

AI-powered time estimates based on task type and complexity. Plan realistic days.
